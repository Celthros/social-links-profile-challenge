# Frontend Mentor - Social links profile solution

This is a solution to the [Social links profile challenge on Frontend Mentor](https://www.frontendmentor.io/challenges/social-links-profile-UG32l9m6dQ). Frontend Mentor challenges help you improve your coding skills by building realistic projects.

## Overview
This project is a solution to the Social Links Profile challenge from Frontend Mentor. The goal is to build a responsive profile card that displays social media links, with a focus on accessibility and interactive states. The solution uses semantic HTML, modern CSS techniques (including custom properties and SASS), and follows a mobile-first workflow. The project demonstrates best practices in layout, styling, and responsive design.


### The challenge

Users should be able to:

- See hover and focus states for all interactive elements on the page
